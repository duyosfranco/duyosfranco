Invitaciones digitales

Ejemplos navegables y listos para compartir

Sumá tus propios diseños agregando nuevos index.html. Cada tarjeta se construye automáticamente desde la lista de invitaciones de abajo.

¿Cómo agrego más invitaciones?

  1. Guardá tu nuevo index.html dentro de una carpeta, por ejemplo invitaciones/boda-azul/index.html.
  2. Editá la sección invitationData al final de este archivo para sumar el título y el enlace.
  3. La tarjeta aparecerá automáticamente con el mismo estilo turquesa.

Ventajas para tus clientes

  • Acceden desde móvil o desktop sin descargar archivos.
  • Compartible por WhatsApp, email o código QR.
  • Animaciones suaves y tipografía legible.
